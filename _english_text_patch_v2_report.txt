Slimefun English Text Patch v2

What this patch does:
- Restores SlimefunItems.java to the China fork code structure, then replaces item names/lore with English text from the English Slimefun source where IDs match.
- Translates core item lore builders: machine tier/type, speed, buffer, material, hunger, range, uses, radiation level.
- Translates two hardcoded item setup names: Birthday Cake and Piglin Trading.
- Keeps item IDs, recipes, machine logic, and source structure from the China fork.

Important:
- This is safer than replacing the whole official SlimefunItems.java, because the whole-file replacement caused compile errors in this fork.
- Some Chinese text may still remain in admin logs, updater messages, storage errors, Android script menus, or China-fork-specific features.
